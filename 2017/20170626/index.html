windows下给Gvim安装NERDTree目录树插件

发表于 2017-06-26   |   分类于 Git

最近在windows下想搞个vim玩玩 最后折腾了好久也算配置的差不多了。 网上有各种插件和配置都可以找到自己需要的，这里只说下NERDTree这个目录树插件的安装吧。这个插件就是在vim编辑器的左边列出一个目录结构就是我们平时看到的那种

安装方法

1.可以去NERDTree的 Github 上去下载 贴个链接 NERDTree
2.下载后解压出来会有大概如下目录

3.然后进入vim的运行目录 我的是D:\Gvim\vimfiles 然后把plugin和doc文件夹下的文件复制到vim的运行目录相同名字的文件夹内
4.最后重启vim 可惜我报错了！！ 缺少了变量 无奈之下把解压的文件夹内的 autoload 文件内的 nerdtree目录和nerdtree.vim复制到了vim的运行目录内对应的文件夹中 以及lib和nerdtree-plugin文件夹一并复制到运行目录下 重启 OK

快捷设置

打开vim后可以在命令行模式下 通过命令 :NERDTree来启用左侧目录树 关闭直接输入 q
另外也可以通过配置快捷键来设置
1.找到你本地的vim安装目录下的配置文件_vimrc
2.在最下面添加如下代码
3.map <F8> :NERDTreeToggle<CR>
4.以后就可以通过快捷键 F8 来展开目录树了
5.重启OK

简单使用

默认打开的Nerdtree会列出桌面的内容，可以通过命令:NERDTree dir_path的方式来切换要展开的目录
eg:
:NERDTree D:\
这样他就会显示D盘的目录结构了
更多的命令和用法可以通过命令 ? 来获得详细实用方法介绍 同样是用?来退出使用介绍
