Win10下 StarUML 3.0 License激活

发表于 2018-10-11   |   分类于 网络编程

之前前端调用支付 判断最终的支付结果 一直都是采用的轮询的方式（反正用户量小的要死） 总感觉不是太好 所以准备重新优化下

然后看了微信的支付时序图 想模仿画个差不多的玩意就 拿出了 StarUML 结果发现过期了 然后下载了个最新的 http://staruml.io/ 又要激活。 身为一个无良 「盗版狗」 当然是要免费激活了

搜索一番终解决 真是佩服这些个大神

激活

首先下载 app.asar 文件

链接：https://pan.baidu.com/s/1wDMKDQkKrE9D1c0YeXz0xg 密码：y65m

备用链接：https://pan.baidu.com/s/11Sm3ASGABNA6KZiFL8Kwuw 密码：xlsh

然后找到你的 StarUML 的安装路径 我的是 C:\Program Files\StarUML\resources 然后其中有一个 app.asar 文件直接用下载好的文件替换掉（先备份下原来的文件吧） 只替换这一个文件即可

然后重新启动 StarUML 进入 选择 Help/Enter License Key 会提示你 Your already have a valid license 说明已经成功激活了
